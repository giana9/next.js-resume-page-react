# Next.js Resume Page (React)

This project showcases my resume. I demonstrated using JSX structure, component rendering, custom CSS styling, and running a local development environment in VS Code.

---
## Technologies Used

- **Next.js** — Framework used to build the resume page  
- **React / JSX** — Component‑based structure and rendering  
- **JavaScript** — Logic and page structure  
- **CSS** — Custom styling for layout and formatting  
- **Node.js & npm** — Development environment and package management  
- **Visual Studio Code** — Code editor used throughout the assignment  

---

---

## How to Run the Project

1. Open the project folder in VS Code  
2. Install dependencies:

```
npm install
```

3. Start the development server:

```
npm run dev
```

4. Open your browser and go to:

```
http://localhost:3000
```

The résumé will load and update automatically as changes are made.
